Employee Management System

About the Project

A comprehensive Employee Management System built with Spring Boot backend and React frontend.

Features:

  • User Authentication and Authorization
  • Employee Management
  • Event Management
  • Booking System
  • Session Management
  • QR Code Integration

Technologies Used:

  • Backend: Spring Boot, Java, JWT Authentication
  • Frontend: React, Vite, Modern JavaScript
  • Database: SQL with JPA/Hibernate
